  • Переход в другие папки

    Запомнить! cd - Change Directory

    Для перехода используется команда cd, после которой указывается путь к нужной папке.
    Если целевая папка находится внутри текущей, то в качестве пути достаточно указать ее имя:

    Запомнить! При вводе названий папок не обязательно писать полные имена, можно пользоваться автодополнением – достаточно набрать несколько первых символов и нажать клавишу Tab.

    Запомнить! ls - List Directories

    Для просмотра содержимого папки нужно ввести команду ls

    Переход в родительский каталог

    Что-бы перейти в родительскую папку, существует команда: cd ..

    Можно подниматься сразу на несколько каталогов вверх по файловой системе: cd ../../../

    Знак ~ (тильда) – это псевдоним, указывающий на домашний каталог пользователя.

    Запомнить! С помощью команды cd ~ - можно с любой директории перейти в домашний каталог пользователя.

  • Находясь в любой папке, можно узнать полный путь к ней от корневого каталога.

    Для этого используется команда pwd (аббревиатура от Print Working Directory)
